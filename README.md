A Discord Bot designed to run on a Raspberry Pi 4.  
List of Features:
- YouTube Video/Audio downloading
- search wayback machine
- helpful dev features (show ip, sending messages, show server status)
- fun commands like viewing random 4chan link, search internet for random screenshot
- etc.


<br>
To run it you need to add a .env with the follwing layout:

```
DISCORD_TOKEN=...
YOUTUBE_API_KEY=...
```
